Run the project's test suite to verify the implementation is working correctly.

1. Detect the project's test framework (pytest, npm test, cargo test, go test, etc.)
2. Run the appropriate test command
3. Report test results, including:
   - Number of tests passed/failed
   - Any failures with clear error messages
   - Code coverage if available

If no tests exist, report this and suggest adding tests.
